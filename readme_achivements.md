# **Баллы**



* баллы по шкале от 1 до 10 за решенный хакатон
* баллы за победу

 	- 1 место (10 баллов)

 	- 2 место (7 баллов)

 	- 3 место (5 баллов)



# **Ачивки**



* 0 баллов - Первый шаг
* 10 баллов - Постоянный гость
* 25 баллов - Ветеран
* 50 баллов - Железный хакер
* 100 баллов - Хакатон-марафонец



# **Таблицы**



## Таблица: achievements (шаблоны ачивок)



|Поле|Тип|Описание|
|-|-|-|
|id|VARCHAR|Первичный ключ|
|name|TEXT|Название ачивки|
|description|TEXT|Описание ачивки|
|icon|TEXT|Ссылка на иконку ачивки|
|value|INT|Количество баллов для достижения|

# 

## Таблица: user\_achievements (ачивки у пользователя)



|Поле|Тип|Описание|
|-|-|-|
|user\_id|VARCHAR|ID пользователя которому принадлежать ачивки|
|achievement\_id|INT\[]|Список с ID ачивок которые есть у юзера|

# 

## Таблица: user\_activity (активность пользователя)



|Поле|Тип|Описание|
|-|-|-|
|user\_id|VARCHAR|ID пользователя|
|visited\_hackathons|INT|Количество посещенных хакатонов|
|points|INT|Общее количество баллов полученных в хакатонах|
