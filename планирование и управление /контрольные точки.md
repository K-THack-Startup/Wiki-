# Майлстоуны проекта

## Milestone 1. Релиз 1 (MVP)
**Результат:**
*   Готова главная страница с карточками организаторов.
*   Регистрация участников.
*   Список участников.
*   ЛК студентов (мои команды и проекты).
*   Расписание событий.
*   Карточка событий (воркшоп, хакатон, митап, тренировка).
*   Админ-панель.

## Milestone 2. Релиз 2
**Результат:**
*   Галерея мероприятий.
*   Возможность управления командами.
*   Добавление бейджей участникам.
*   Заявки партнера со статусами.
*   Шаблоны для задач.
*   ЛК партнера.
*   Телеграм-бот с уведомлениями.

## Milestone 3. Релиз 3
**Результат:**
*   Фильтры для проектов (технология, дата, хакатон, команда).
*   Уведомления организаторам на почту.
*   Оценка и выбор задачи от партнера.
*   Хронология побед: год - хакатон - команда - проект.
*   Фото дипломов и кубков.
*   Ссылки на проекты-победители.
*   Страницы с хакатонами.
*   Регистрация на хакатон через наш сайт.
*   Оценка проектов через платформу.
