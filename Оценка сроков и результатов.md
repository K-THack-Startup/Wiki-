# Оценка сроков и результатов

## 1. Спринты

*   **Подготовка (Спринт 0) - 1 неделя**
    *   Техническая подготовка, прототипирование, планирование.
    *   Готовность команды и среды к разработке.

*   **Ядро платформы (Спринты 1-2) - 2-5 неделя**
    *   Разработка базовой архитектуры, регистрация, профили.
    *   Работающая система аутентификации и ЛК.

*   **Поиск команд (Спринты 3-4) - 6-7 неделя**
    *   Формирование механизма поиска, приглашений, создания команд.

*   **Завершение MVP (Спринты 5-6) - 8-9 неделя**
    *   Тестирование и финальная подготовка к запуску.

## 2. Декомпозиция

### Спринт 1-2 (Неделя 2-5): Ядро системы
*   API аутентификации **(5 дней)**
*   Интерфейс регистрации и входа **(4 дня)**
*   Профили пользователей **(6 дней)**
*   Личный кабинет **(3 дня)**
*   API профилей **(2 дня)**

### Спринт 3-4 (Неделя 6-7): Поиск команд
*   Интерфейс поиска **(5 дней)**
*   Бэк поиска **(4 дня)**
*   Система приглашений **(6 дней)**
*   Модель команды + чат **(5 дней)**
*   Управление командой **(4 дня)**

### Спринт 5-6 (Неделя 8-9): Запуск
*   Расписание событий **(4 дня)**
*   Админ-панель **(3 дня)**
*   Тестирование **(3 дня)**
*   Деплой **(2 дня)**
*   Запуск **(1 день)**

## 3. Планируемые результаты по этапам

### По завершению первого месяца (4 недели)
*   Рабочая система регистрации и аутентификации.
*   Личные кабинеты пользователей с заполнением профилей.
*   Базовая структура данных.
*   Готовность ~40% функционала MVP.

### По завершению 2 месяца (7 недель)
*   Полнофункциональный поиск пользователей по фильтрам.
*   Система отправки/принятия приглашений в команды.
*   Механизм создания виртуальных команд.
*   Базовая чат-система для команд.
*   Готовность: ~80% функционала MVP.

### К финальному запуску (9 недель)
*   Работающая админ-панель.
*   Расписание событий, возможность добавления события.
*   Платформа развернута на продакшн-сервере.
*   Первичный контент.
*   MVP готов к запуску.s
